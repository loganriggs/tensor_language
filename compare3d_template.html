Where each structure is stored: top-3 PCs of the residual stream, per graph family

One panel per graph structure (torus and ER were never trained on). Each panel's dropdown switches between the multi-family bilinear model (the positive organizer), the single-family grid specialist, the multi-family softmax champion (the anti-organizer), and the true spectral layout as reference. Big circles = per-node mean representations; small dots = individual per-position residual states in the same PC basis; gray lines = true edges. Drag to rotate, scroll to zoom; x/y/z pick PCA dims; slider sets context length.

What to look for: in the multi-family bilinear view, the ring becomes a circle, the directed ring a circle, the grid a bent sheet, the cylinder a tube, the torus (never trained!) a ring-of-rings, and the tree/3-regular/ER graphs arrange with neighbors adjacent — compare each against the true spectral reference in the same panel. Flip to the grid specialist or the softmax model to see the same documents stored without local geometry (clouds overlapping, edges crossing).
